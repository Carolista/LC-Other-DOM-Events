Mouseover and Mouseout

This button requires two separate handlers. One is listening for the mouse to cross the boundary onto the button, and the other one listens only for the mouse to cross back over the boundary to the rest of the page. Note that if your goal is to affect only the item you are moving over, it's more common to use the pseudo-element :hover in your style sheet. The second button demonstrates a situation where handling it in the DOM is the only way to go.

Mousedown and Mouseup

This button also requires two separate handlers. It is different from a click event because one handler listens only for the press part of a click, and the other one listens for the release part of the click. Try it out! Start to click on the button, but don't release your mouse right away. Compare this to the behavior of the regular click button above, which silently acknowledges the pressing of the mouse, but waits for the release before changing the color.
